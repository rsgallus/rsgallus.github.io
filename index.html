Projects

Pilot Biometrics

For Senior Design class, my team was tasked with developing a pilot biometrics system to capture ECG waveforms. The goal of the system is to monitor pilot health in flight and alert pilots when they are at risk of falling unconscious. We were given an ARM M7 microprocessor card, a medical-grade ECG sensor, and wireless electrodes. With this, my team was asked to set up a UNIX environment, design a filter for the ECG to achieve a high degree of fidelity, write an algorithm for analyzing the sensor data, write device drivers for the hardware, design circuit components to connect hardware, and develop a mechanical harness to mount the sensors.


Plants by WebSphere

One of my major tasks while working for IBM was to completely overhaul an outdated sample application called Plants by WebSphere. This involved removing the old Java Server Pages and Java Server Faces design, and rebuilding the application as a RESTful web service. In addition to these changes, I also redesigned the user interface to scale to display size. The project was done completely open source, so you can see the latest changes on GitHub.

I was in charge of the mobile app, which I built using Apache Cordova to create native Android and iOS applications. At the end of the semester, our design was nominated for best project.

Plow Tracker Map

At the Iowa Department of Transportation, I developed a web application that plots the locations of Iowa DOT snow plows in real time. Each snow plow was fitted with a GPS unit that sends data to an Oracle Spatial database. Using location data and the ESRI ArcGIS Online API, I developed an interactive JavaScript web application. The application allows anyone to view the locations of snowplows and even view dash-camera footage of snow plows as they are working, which helps travelers predict the condition of snowy roads.
